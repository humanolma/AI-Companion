## 一、项目概览

| 项目 | 说明 |
|------|------|
| 名称 | AI 虚拟伴侣（AI Virtual Companion） |
| 定位 | 有个性、有记忆、有情感的对话型 AI Agent |
| 目标受众 | GitHub 开源社区 / 面试官 |
| 技术栈 | Python + LangChain + DeepSeek API + FastAPI |
| 核心亮点 | 角色人设 / 双层记忆系统 / 情感感知 / 流式输出 / 自研 Web UI |

### 核心功能规划

1. **角色人设系统** — 可自定义伴侣的性格、背景、说话风格
2. **记忆系统** — 短期上下文（已实现）+ 长期记忆（已实现，ChromaDB）
3. **情感感知** — 识别用户情绪，调整回复语气和内容（已实现）
4. **Web 界面** — 自研 FastAPI + 原生前端，玻璃拟态设计（已完成）

## 四、遇到的问题

| 日期 | 问题描述 | 解决方案 | 状态 |
|------|----------|----------|------|
| 2026-07-04 | Gradio 6.x Chatbot 格式错误 | 设置 `type="messages"` 并使用 dict 格式 | ✅ |
| 2026-07-04 | UI 单调无头像 | 创建 SVG 头像并配置到 Chatbot | ✅ |
| 2026-07-04 | langchain 1.x embeddings 导入路径变化 | 自实现 `LocalEmbeddings` 包装类，不依赖 langchain 版本 | ✅ |
| 2026-07-04 | DeepSeek 不支持 embeddings API | 使用本地 sentence-transformers 模型做向量化 | ✅ |
| 2026-07-04 | 缺少 langchain-chroma 包 | 安装 `langchain-chroma` 并加入 requirements.txt | ✅ |
